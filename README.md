# Evidence Based AI Agent

This example demonstrates an AI agent designed to make evidence-based decisions rather than assumptions. It simulates a simple diagnostic system that only provides a confident diagnosis when sufficient evidence (symptoms) is present. If evidence is insufficient, the agent explicitly states this and suggests what additional information would be helpful, preventing potentially harmful guesses.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
